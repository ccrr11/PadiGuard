# PadiGuard
# Rice Disease Diagnostic AI 🌾
This project uses **Gemini 3 Flash** to diagnose rice diseases from images.

### How it Works
The model is configured with a specific **System Instruction** that converts image 
analysis into a structured JSON format for our app.

### Disease Coverage
- **Rice Blast**: Identified by spindle-shaped lesions.
- **BLB**: Identified by wavy yellow streaks.
- **Brown Spot**: (Coming Soon) Identified by circular "sesame seed" spots.
